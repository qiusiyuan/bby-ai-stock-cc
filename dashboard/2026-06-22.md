### III. 核心问题：Hyperscaler 在烧多少钱

虽然今天的文章没给数字，但行业共识级别的 2026 AI capex（数据来自之前的财报指引，标 *approx*）：

```compare

```

**为什么市场现在才"突然"怀疑？**

- 之前的板块假设：AI capex → 未来 AI revenue（chatGPT 类应用爆发） → 估值合理
- 现实证据：
  - ChatGPT 收入扩张快，但 **它的收入流向 OpenAI / Anthropic，不是 hyperscaler**
  - Hyperscaler 的"AI 自有产品"（Bedrock / Vertex / Copilot）收入贡献远小于 capex
  - **Capex 转折旧周期 4-6 年** —— 即使明天就停止投入，未来 4 年的财务报表都要承担每年 ~$80B 的折旧吞噬 OPEX
- 市场情绪转折点：今天三件事叠加（人才流失 + 没有 ROI 数据 + Fed 鹰派化），让"_justify_"这个词第一次冒头。

```callout warn
**关键洞察**：这不是说 AI 错了，而是说"AI 应用层赚钱的人 ≠ 上游 / 中游"。
- OpenAI / Anthropic 是赢家（但没上市）
- NVDA / TSM / SK Hynix / MU 是赢家（已经反映在股价里）
- **Hyperscaler 是中间商，可能既不是 winner 也不是 loser** — 它们烧的钱让 AI 革命发生，但自己未必能 capture 大部分 value
```
